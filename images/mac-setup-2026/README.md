# Screenshots for the 2026 Mac setup post

Drop the screenshots referenced from the post into this directory. Suggested captures (filenames match the post):

- `desktop-overview.png` — full desktop with Warp + Cursor/VSCode + Claude side-by-side
- `warp-terminal.png` — Warp showing AI command suggestion or a workflow block
- `cursor-claude.png` — Cursor or VSCode with Claude / inline AI panel open
- `claude-desktop.png` — Claude desktop app with a project / context loaded
- `chrome-profiles.png` — Chrome profile switcher showing your work / personal / founder profiles
- `obs-scene.png` — OBS with your scene collection (cam-only, screen + cam, screen-only)
- `keycastr-demo.png` — KeyCastr overlay during a recording
- `brewfile-output.png` — terminal output of `brew bundle --file=~/Brewfile`

Recommended size: 1600px wide max, PNG, optimize with `pngquant` or similar before committing.
